═══════════════════════════════════════════════════════════════════════════════
  FEATURE 2: DEALER SYSTEM 🤝
═══════════════════════════════════════════════════════════════════════════════

WHAT IT IS:
  Three-tier access control system:
  • ADMIN: Full control (users, dealers, products, orders)
  • DEALER: Add products, manage assigned customers
  • USER: Regular shopping (sees approved products)

HOW IT WORKS:
  1. Admin creates dealer account
  2. Admin assigns customers to dealer
  3. Dealer adds products (pending approval)
  4. Admin approves products
  5. Products visible to all users
  6. Dealer manages customer orders
  7. Dealer earns commission
