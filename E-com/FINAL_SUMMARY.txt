                            🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

ISSUE: PWA not installing
FIX: Ensure HTTPS or localhost, check manifest.json accessible

ISSUE: Service worker not registering
FIX: Check sw.js in public/ folder, check browser console

ISSUE: Icons not showing
FIX: Create icon-192.png and icon-512.png in public/

ISSUE: Theme not switching
FIX: Verify CSS variables defined, check data-theme attribute

ISSUE: Components not found
FIX: Ensure all files in src/components/ folder

ISSUE: Helpers not working
FIX: Check src/utils/helpers.js exists, verify imports
